## Run Locally

Clone the project

```bash
  git clone https://github.com/tobiimdp12/HeroApp.git
```

Go to the project directory

```bash
  cd HeroApp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
